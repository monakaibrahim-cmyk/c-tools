\doxysubsection{Detailed Description}
File creation and timestamp modification utility. 

A cross-\/platform command-\/line utility inspired by the Unix \textquotesingle{}touch\textquotesingle{} command. Creates new files and modifies access/\+modification timestamps on existing files. Designed specifically for Windows systems with support for various timestamp modification options.
