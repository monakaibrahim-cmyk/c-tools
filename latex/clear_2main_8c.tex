\doxysubsection{Detailed Description}
Clear screen utility. 

This utility clears the terminal screen by printing ANSI escape sequences. It provides a simple cross-\/platform way to clear the console output.

Main entry point for the clear command. 

This function clears the terminal screen by sending the ANSI escape sequence for cursor home position (\textbackslash{}033\+[H) followed by the clear screen sequence (\textbackslash{}033\+[J). This is compatible with most modern terminal emulators.
