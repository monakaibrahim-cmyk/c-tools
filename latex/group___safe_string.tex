\doxysubsection{Detailed Description}
Compiler-\/agnostic safe string operation macros. 

Provides macros that wrap secure versions of string functions. Uses MSVC \+\_\+s variants when compiled with MSVC, otherwise falls back to standard C99/\+\+C11 functions with manual null-\/termination.
