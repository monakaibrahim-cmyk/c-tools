\doxysubsection{Detailed Description}
Directory tree visualization utility for Windows. 

@description A Windows command-\/line utility that displays the directory tree structure starting from a specified path. It recursively traverses directories and presents the hierarchy in an ASCII tree format, similar to the Unix \textquotesingle{}tree\textquotesingle{} command.

Features:\+
\begin{DoxyItemize}
\item Recursive depth-\/first directory traversal
\item ASCII tree visualization with branch characters
\item File and directory counting
\item Volume information display (drive, serial number, filesystem type)
\item Colored terminal output using ANSI escape sequences
\end{DoxyItemize}

@usage tree \+[directory\+\_\+path]\+

@examples tree \# Display tree of current directory tree C:\+\textbackslash{}\+Users \# Display tree of specified path

\begin{DoxyNote}{Note}
Requires Windows API (WIN32\+\_\+\+FIND\+\_\+\+DATA, Find\+First\+File/\+\+Find\+Next\+File)
\end{DoxyNote}
